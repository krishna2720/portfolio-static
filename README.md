# Portfolio Website

A clean, modern, and fully responsive portfolio website showcasing my journey as a Full Stack Developer and Competitive Programmer. Built with a focus on performance, clean structure, and elegant design.

## 🚀 Features
* **Responsive Design:** Optimized for all screen sizes (Desktops, Tablets, and Mobile).
* **Dynamic Fallbacks:** Automated profile avatar generation using DiceBear API if local assets fail to load.
* **Tech Showcase:** Highlights core competencies in MERN stack development, FastAPI, SQL, and DSA.
* **Project Gallery:** Structured display of featured projects like Spatial Bench, StudyNotion, and shop_krish.
* **Direct Access:** Seamless integration for downloading my latest technical resume and quick contact links.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Typography & Icons:** Google Fonts (Poppins), Font Awesome 
* **Design Pattern:** Clean semantic HTML structure paired with modern flexbox/grid layout systems.
